Basic Functionality

In the "Web 2.0" world editing is an essential feature of any grid component. The ko-grid supports editing via the editing extension. To try it out, select a cell and start typing.

Note: You have full control over the editor and how changes are saved.

Note: This is the very first iteration of editing support. If you have suggestions for improvements, feel free to file a feature request.
