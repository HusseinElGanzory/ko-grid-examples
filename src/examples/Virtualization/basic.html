Basic Functionality

The virtualization extension enables row virtualization for the ko-grid. Virtualization is a common optimization when displaying many records, only a few of which fit on the screen at one time. Only those which will be visible in the view port are added to the DOM. This increases performance while reducing memory consumption. Also, when a remote data source is used, it reduces the records which need to be requested from the server.
