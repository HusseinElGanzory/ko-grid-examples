Column Resizing

Naturally, the full screen extension can be combined with column resizing. However, what happens when you toggle full screen mode on and off? Just what the user would expect: The grid keeps track of column widths in and out of full screen mode separately. Each time it is toggled, the previously configured value is restored.

Note: These column widths are stored in localStorage under the grid's id. In other words, the column widths are remembered across page reloads and sessions.

Known Issue: Entering full screen mode does not always work properly in WebKit and Blink based browsers. You can still see the expected result by resizing the window/view port. This issue is tracked on github.
