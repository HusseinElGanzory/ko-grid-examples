Fixed Height

If you want to limit the grid's vertical size, you can specify a height for it. Any height value is allowed – absolute as well as relative values are supported. If the space is not sufficient to fit the data, the table body will become scrollable.

Note: Don't forget to add the class fixed-height to your grid, if you want to specify a fixed height. Otherwise your browser will display a horizontal scroller to make up for the space taken by the vertical one.
