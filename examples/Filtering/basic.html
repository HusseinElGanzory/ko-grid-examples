Basic Functionality

The filtering extension displays a row of search boxes below the column headers. Entering a text into a search box will filter rows based on whether they match the search term in that column.

Search terms are interpreted as follows.

  • Iff the search term contains no upper case letters, a case insensitive search is performed.
  • An asterisk (*) matches any substring.
  • A search term containing no asterisks implies
    • a substring search and
    • a loose "hump search" (CR is matched by "Czech Republic" and "Costa Rica").
