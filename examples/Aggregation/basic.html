Basic Functionality

The aggregation extension calculates various aggregate values and displays them in the table footer. Supported aggregates are the sum, the average as well as the minimum and maximum value.

Example: Countries and dependencies by population

Source
Wikipedia

Example configuration

aggregate: [
    { population: 'sum', percentage: 'sum' },
    { population: 'minimum' },
    { population: 'average', percentage: 'average' },
    { population: 'maximum' }
]
